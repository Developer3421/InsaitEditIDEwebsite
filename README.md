# Insait Edit — Landing Page

A Next.js landing page for the **Insait Edit** IDE product line, built in a purple-orange style.

![Insait Edit Landing Page](https://github.com/user-attachments/assets/f3534484-e72a-4220-ba1c-89f0ea165021)

## About

**Insait Edit** is a product line of modern, open-source IDEs:

| Product | Status |
|---|---|
| Insait Edit IDE C# | ✅ Available |
| Insait Edit IDE PHP | 🚧 In Development |
| Insait Edit IDE Web | 📅 Planned |

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Language**: TypeScript
- **Font**: System sans-serif (Arial, Helvetica)

## Building for Production

```bash
npm run build
npm run start
```
